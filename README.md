# Instagram Clone

This is a web app that I was building for more than 2 months to practice my skills. As the name says, it's clone of
the pretty popular social media platform Instagram.

## Features
- Login and Signup 
- Follow and Unfollow users
- Like/Unlike a post
- Comments
- Get suggested users
